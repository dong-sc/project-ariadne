# Project Ariadne

> Every labyrinth yields to a thread.

Project Ariadne is a browser-based studio management game about attention, pressure, trust, and building reliable workflows.

The player begins by manually rescuing a chaotic photography studio. Over time, the optimal strategy shifts from clicking faster to designing a system that prevents fires before they start.

## North Star

**建立系統，而不是等待奇蹟。**

## Current playable slice

Sprint 1 focuses on making chaos visible through the room itself:

- four linked workstations
- click as attention allocation
- visible queues, warning states, and local movement
- deadlines and recoverable stage failure
- a calmer room after successful delivery

## Stack

- TypeScript
- Vite
- PixiJS
- Vitest

- GitHub Pages
